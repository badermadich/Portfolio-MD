Structure du Répertoire
-------------------

css\                - Fichiers CSS de styles pour le modèle.
images\             - Images utilisées dans le modèle.
scripts\            - JavaScript utilisé dans le modèle.
styles\             - Fichiers source SCSS pour les feuilles de style CSS. Disponibles uniquement dans les versions payantes.
favicon.ico         - Favicon fourni en tant que substitut pour le modèle.
index.html          - Page HTML principale pour ouvrir le modèle dans le navigateur.
